# Simple Todo App

This is an application that allows the user to create a list of todo tasks. It also has a checkbox so they can be marked as completed along with an area to place the due date for when the task needs to be done.

## Functionality

When entering the information into the add ToDo, it will scan the input information to ensure that it is valid. There is a delete button that will delete the task once completed.

## Technology

I separated my javascript into different classes and enabled them to interact together.
